# Assessment Framework

## Purpose
Supports readiness assessment using practice scores and weak topic evidence.

## Readiness Logic
A learner should not be considered exam-ready without evidence from practice scores or assessment results.

## AI-102 Readiness Target
Recommended target practice score: 80 or above.

## Reassessment
Learners below target should complete remediation and retest before scheduling the exam.
