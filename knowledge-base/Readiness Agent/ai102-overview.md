# AI-102 Certification Overview

Certification: AI-102
Name: Azure AI Engineer Associate

## Purpose
AI-102 validates the ability to design, implement, secure, monitor and evaluate Azure AI solutions.

## Key Capability Areas
- Azure AI Services
- Azure AI Foundry
- Azure AI Search
- Prompt Engineering
- Retrieval-Augmented Generation
- AI Agents
- Responsible AI
- AI Solution Design

## Recommended Foundation
AI-900 or equivalent AI fundamentals.
