# Readiness Framework

## AI-102 Readiness Thresholds
Ready: practice score 80 or above.
Nearly Ready: practice score 70 to 79.
Requires Structured Preparation: practice score 60 to 69.
Not Ready: practice score below 60.

## AI-102 Common Weak Areas
- Azure AI Search
- Prompt Engineering
- AI Agents
- Retrieval-Augmented Generation
- Azure AI Services integration
- Responsible AI
- RBAC and security configuration

## Decision Rules
Support when readiness score is 80 or above.
Conditional Support when score is between 60 and 79.
Oppose when score is below 60.
