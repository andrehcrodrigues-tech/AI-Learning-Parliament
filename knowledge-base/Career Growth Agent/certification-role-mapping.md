# Certification Role Mapping

## AI-900
Purpose: AI fundamentals and basic AI literacy.

## DP-900
Purpose: data fundamentals and foundational data literacy.

## DP-700
Purpose: Microsoft Fabric data engineering.

## AI-102
Purpose: Azure AI solution implementation and AI engineering capability.

## Infrastructure Project Manager to AI Transformation Lead
Preferred certification: AI-102.
Alternative: AI-900 first if AI foundations are weak.

## Senior Network Engineer to Principal Network Architect
AI-102 alignment: Low.
Alternative certifications should be considered for network architecture paths.
