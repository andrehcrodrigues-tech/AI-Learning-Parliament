# Architecture

```text
Manager-Agent
Career-Growth-Agent
Readiness-Agent
Capacity-Agent
Future-Skills-Agent
        ↓
Critic-Agent
        ↓
Speaker-Agent
```

The workflow starts with a learner profile and certification question.

The five stakeholder agents evaluate the request independently using their Knowledge Base sources.

The Critic Agent validates consistency, contradictions, missing evidence and risk.

The Speaker Agent produces the final AI Learning Parliament Decision.
